Accessibility

Our website conforms to the latest web design standards to help accessibility. All the pages meet W3C standards for HTML and CSS.

Web Browser Compatibility

Our website has been tested and work on Firefox and Internet Explorer.

Visually Impaired

To support users that use a screenreader to navigate websites, we have ensured thew following:

  • All links have title tags
  • All images have alternative text
  • The CSS has been seperated to ensure HTMl code is light and clean
  • The text colour has enough contrast with the background colour to make it easy to read

Restricted Mobility

To support users with restricted mobilty, we have ensured thew following:

  • The main menu links use a large font
  • A Sitemap has been included to provide navigation to all pages form one place
